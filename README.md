# HOW TO RUN
1. Install XAMPP (https://www.apachefriends.org/)
2. Turn on "Apache" and "MySQL"
3. Create a new database schema named "todo_db" here (http://localhost/phpmyadmin/)
4. Open the project in IntelliJ and hit run!
